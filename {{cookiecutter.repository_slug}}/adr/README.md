# ADR

В каталоге `adr/` храните только реальные архитектурные решения.

Правила:

- используйте шаблон `templates/adr.md`;
- создавайте ADR только при наличии явного архитектурного выбора;
- не держите в каталоге демонстрационные решения.

Архитектурные решения фиксируются отдельными ADR и ссылаются из feature-файлов и профильных разделов документации.
