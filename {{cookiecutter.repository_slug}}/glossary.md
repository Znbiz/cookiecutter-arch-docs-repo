# Глоссарий

## Подписка

Долгоживущая договорённость между пользователем и продуктом на получение услуги по выбранному тарифу.

## Активация подписки

Переход подписки в состояние `active` после успешной оплаты и выполнения всех доменных проверок.

## Renewal

Продление действующей подписки на следующий оплаченный период.

## Grace period

Период после неуспешного списания, в течение которого сервис может сохранить ограниченный доступ.
