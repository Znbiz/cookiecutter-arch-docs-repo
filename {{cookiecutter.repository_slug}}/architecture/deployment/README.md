# Deployment

Раздел фиксирует runtime topology, сетевые ограничения, конфигурацию, rollout, rollback и параметры стендов.

Для каждого стенда должна быть явно зафиксирована информация:

- адреса и точки входа;
- внутренние endpoint'ы, если они важны для эксплуатации;
- runtime-ресурсы: реплики, CPU, память, класс БД, класс брокера;
- owner или команда сопровождения стенда.
