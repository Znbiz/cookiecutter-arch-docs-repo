# Configuration

- адреса внешних и внутренних endpoint'ов;
- настройки интеграций, topic, queue, consumer groups;
- feature flags для rollout новых сценариев;
- параметры ретраев, таймаутов и circuit breaking;
- параметры ресурсов, если они управляются конфигурацией;
- ссылки на секреты, vault paths или secret scopes без раскрытия самих секретов.
