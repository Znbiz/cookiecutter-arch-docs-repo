# Contract Tests

Добавляйте сюда contract-тесты и compatibility checks.

Правила:

- фиксируйте покрытие для API, event и data contracts;
- для breaking changes обязательно описывайте миграционный сценарий;
- для оформления кейсов используйте шаблон `.arch-docs/templates/test-case.md` с `Type: contract`.
