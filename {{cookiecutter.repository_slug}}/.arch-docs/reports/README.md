# Reports

В этот каталог можно сохранять детерминированные отчёты локальных режимов:

- `standard-audit.md`
- `standard-migration.md`
- `bootstrap-gap-report.md`

Отчёты можно генерировать локально через `.arch-docs/scripts/audit-standard.sh`,
`.arch-docs/scripts/migrate-standard.sh` и `.arch-docs/scripts/bootstrap-repo.sh`.

Если отчёт создаётся только для локальной проверки, его можно не коммитить.
